# data-clean-mini-project
# مشروع تنظيف بيانات اليوم 15
 

## خطوات التنظيف
1. العمر (Age): 
   - حولناه إلى رقم (float)  
   - عوضنا القيم الناقصة بالوسيط  
   - قصينا القيم الكبيرة جدًا عند 99٪  

2. الدخل (Income):  
   - حولناه إلى رقم (float)  
   - عوضنا القيم الناقصة بالوسيط  
   - قصينا القيم الكبيرة جدًا عند 99٪  

3. المدينة (City):  
   - شلنا الفراغات  
   - حولنا كل الحروف لصغيرة  

4. تاريخ التسجيل (Signup_time):  
   - حولناه لتاريخ  
   - ضبطنا المنطقة الزمنية UTC  
   - أي تاريخ غير صالح يصبح NaT  

## إضافي
- أعمدة `age_missing` و `income_missing` تحدد الصفوف اللي كانت ناقصة وهي صفوف أضفته أثناء التنظيف 
- الكود يوضح تأثير كل خطوة تنظيف على البيانات
